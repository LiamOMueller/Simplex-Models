# Simplex-Models
 Three and four dimentional relationships reduced using simplex analysis

This project began as an attempt to model 4 dimensional multivariate classroom activity data reported by students.
That project is now published: Liam O. Mueller, Manvir Bamrah, Katherine L. Petrie, Ella Tour, Lisa McDonnell, Stanley M. Lo, and Clara L Meaders (2026) The Fourth Credit Hour: How do we Teach in Biology Discussion Sections? CBE—Life Sciences EducationVol. 25, No. 1
Given simpler 3 dimensional data can also be visualized in a 2D simplex, I have begun adding that work here as well.
